# Nice Digital Clock 

A simple yet stylish digital clock made with **Tkinter** in Python.

## Features
- Displays current time in 24-hour format (HH:MM:SS)
- Shows date (YYYY.MM.DD), day of the week, and AM/PM
- Uses **DS-Digital** font for a retro digital look (falls back to Courier if not installed)
- Dark theme with cyan time display

## Requirements
- Python 3.x
- Tkinter (usually comes with Python)

## Optional: DS-Digital Font
For the best visual effect, download and install the **DS-Digital** font:  
[Download DS-Digital font here](https://www.dafont.com/ds-digital.font)

If the font is not available, the clock will automatically use Courier.
